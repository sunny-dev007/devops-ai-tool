# ⚡ SQL Developer AI Assistant - Quick Start

## 🚀 5-Minute Setup

### 1. Open the Assistant
```
http://localhost:3000/sql-operations
```

### 2. Connect (3 Steps)
1. **Select SQL Server** → Choose from dropdown
2. **Select Database** → Pick your database
3. **Enter Password** → Your database password → Click "Analyze Schema"

### 3. Start Chatting!
```
"Show me all tables"
"Get active users from last month"
"Optimize my slow queries"
```

## 🚨 Need Help?

**Issue:** Can't connect  
**Fix:** Check password, firewall, and database status

**Issue:** No Execute button  
**Fix:** Install sqlcmd: `brew install sqlcmd`

**Issue:** Slow responses  
**Fix:** Schema loaded? Check connection status
