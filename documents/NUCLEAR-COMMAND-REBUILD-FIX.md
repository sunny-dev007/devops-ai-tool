**File:** `services/executionService.js`

**Function:** `cleanWebAppCreateCommand()` - Lines 1078-1154

### **The Nuclear Rebuild Process:**

```javascript
// Step 1: Extract ONLY the 3 required parameters using regex
const nameMatch = command.match(/--name\s+["\']?([^"'\s\\]+)["\']?/);
const rgMatch = command.match(/--resource-group\s+["\']?([^"'\s\\]+)["\']?/);
const planMatch = command.match(/--plan\s+["\']?([^"'\s\\]+)["\']?/);

// Step 2: Extract values
const webAppName = nameMatch ? nameMatch[1] : null;
const resourceGroup = rgMatch ? rgMatch[1] : null;
const plan = planMatch ? planMatch[1] : null;

// Step 3: REBUILD command with ONLY these 3 parameters
const cleanCommand = `az webapp create --name "${webAppName}" --resource-group "${resourceGroup}" --plan "${plan}"`;

// Step 4: Return REBUILT command (original is DISCARDED)
return cleanCommand;
```
